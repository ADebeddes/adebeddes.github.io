Context

In my learning process, I took a look at the different game engines on the market. I learned about the existence of Godot. Given its good reputation and its frequent use in gamejam, I embarked on learning this engine.
For this, I based myself on the work done in the video "How to make a Video Game - Godot Beginner Tutorial" by Brackeys (link at the end of the article)
I added various elements such as a menu, a victory condition as well as changes to the map in order to create a complete first game.
I took advantage of this game to be able to publish my first game on itch.io with a free and playable version of the game on browser.

Gameplay

The game is a 2D platformer in which the player play as a knight. This knight must retrieve the pieces scattered throughout the map. For this, the player will have to dodge enemies and perform precise jumps.
